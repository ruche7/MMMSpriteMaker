■はじめに

『MMMSpriteMaker』は、他のツールで作成したテクスチャアトラスから
MikuMikuMoving用のアクセサリとエフェクトを自動生成するツールです。

出力したアクセサリおよびエフェクトは、設定によりスプライト、ビルボード、
あるいは板ポリゴンとして表示されます。

最新版は私のWebサイトからどうぞ。
- http://www.ruche-home.net/

MikuMikuMoving(略称MMM)はMikuMikuDance(略称MMD)の互換ソフトです。
詳しくはマニュアルページをご覧ください。
- https://sites.google.com/site/mikumikumoving/

===============================================================================
■動作環境

- MikuMikuMoving version 1.2.5.8 以降が動作する環境。
- 頂点シェーダ3.0、ピクセルシェーダ3.0に対応する環境。

展開したファイル群を、フォルダ構成を崩さずに適当な場所に置いてください。
MMMSpriteMaker.exe がツール本体です。

■アンインストール手順

インストール時に配置したファイル群を削除してください。

一度でも使用すると、ローカルアプリケーションフォルダに設定が保存されます。
ローカルアプリケーションフォルダは標準では次の場所になります。

- Windows XP 等
    "C:\Documents and Settings\[ユーザ名]\Local Settings\Application Data"

- Windows 7
    "C:\Users\[ユーザ名]\AppData\Local"

"Local Settings"フォルダや"AppData"フォルダ等は隠しフォルダです。
このフォルダ内の次のフォルダに設定が保存されます。

- "ruche-home\MMMSpriteMaker"

完全にアンインストールするには、これらのフォルダも削除してください。
大したサイズにはならないはずなので、無理に削除する必要はありません。
